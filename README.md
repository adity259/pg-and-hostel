# pg-and-hostel

PGLife is a simple and modern web application that helps students and working professionals find affordable PGs and hostels in different cities.
The platform provides a search feature, city listings, and a responsive user interface for easy accommodation discovery.

---

## 🚀 Features

* 🔎 Search PGs and Hostels by city
* 🏙️ Popular city listings
* 👤 User Signup and Login modal
* 📱 Fully responsive design
* 🎨 Modern UI using Bootstrap
* ⚡ Fast and lightweight website
